# Titanic EDA

## Project Overview
Exploratory Data Analysis on Titanic dataset to understand survival patterns.

## Objective
To analyze which factors influenced passenger survival.

## Dataset
Titanic dataset containing passenger details like age, gender, class, and survival status.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps
- Data cleaning
- Handling missing values
- Visualization
- Pattern analysis

## Key Insights
- Females had higher survival rate
- 1st class passengers had better survival chances
